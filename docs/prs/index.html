Using Github classroom in ICS3U

What I want students to learn

I want students to learn how to code in a collaborative way using industry standard tools.

How will I know when students have learned?

Github classroom is a place to submit code for submission, group review, and auto grading.

The teacher can set up automated tests and encourage commenting on each other's code.

The teacher can get a view of each student's engagement.

Multiliteracy principles and concepts

Students get to be part of the Production edge of the media triangle - how are websites made? What are algorithms?

Learning is presented mostly asynchronously via documentation, videos, and a heavy use of visuals.

Students engage in collaboration via commenting on code.

Why this instead of Google Classroom?

The main strength of this project is it's something I will actually use in the future.
